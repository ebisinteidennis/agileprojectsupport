Add LiveSupport chat widget to your WordPress site for real-time customer support.

== Description ==

LiveSupport Chat Widget allows you to easily add a professional live chat system to your WordPress website. Connect with your visitors in real-time and provide instant customer support.

**Features:**
* Real-time messaging
* File sharing capabilities
* Mobile responsive design
* Easy customization options
* Multiple agent support
* Message history tracking
* Visitor information capture

**Getting Started:**
1. Create an account at [LiveSupport](https://agileproject.site)
2. Get your Widget ID from the dashboard
3. Install this plugin and enter your Widget ID
4. Enable the chat widget and you're ready to go!

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/livesupport-chat/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > LiveSupport Chat to configure the plugin.
4. Enter your Widget ID (get it from your LiveSupport dashboard).
5. Enable the chat widget and choose its position.
6. Save settings and the chat widget will appear on your website.

== Frequently Asked Questions ==

= How do I get a Widget ID? =

You need to create an account at [LiveSupport](https://agileproject.site) and get your Widget ID from the dashboard.

= Can I customize the widget appearance? =

Yes, you can customize colors, position, and messages from your LiveSupport dashboard.

= Does it work on mobile devices? =

Yes, the chat widget is fully responsive and works great on all devices.

= Can I use multiple agents? =

Yes, depending on your LiveSupport plan, you can have multiple agents responding to chats.

= Can I add the widget manually to specific pages? =

Yes, you can use the shortcode `[livesupport]` or the PHP function `livesupport_display_widget()` in your theme.
